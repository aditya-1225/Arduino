# Arduino Code Repository

This repository contains a collection of independent Arduino sketches developed for testing, prototyping, and experimentation with different hardware modules and interfaces.

---

## Contents

- **LED_STRIP_10.ino**  
  WS2812B LED strip controller using the FastLED library and an ATmega328P-based PCB.

- **GPS_Logger.ino**  
  Reads and outputs GPS data from a GPS module using NMEA sentences.

- **clock_oled.ino**  
  Displays time information from an RTC module on an OLED display.

- **nrf_weather_transmission**  
  Weather data from a BMP280 sensor transmitted wirelessly from a transmitter (nRF24L01+PA+LNA) to a receiver (nRF24L01+PA+LNA), which sends the data to a computer for processing.
